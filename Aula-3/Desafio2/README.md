### 6) Dado a seguinte classe:

```java
public class Pessoa {
            String  nome;
            int idade;
}
```

* Encapsule os atributos da classe
* Faça a consistência dos atributos (nome não pode começar com branco e idade tem que ser maior que zero)
* Crie um classe (OperaPessoa) com métodos para executar as seguintes operações:
* Obter um ArrayList de objetos do tipo Pessoa;
* Exibir os nomes e as idades de todas as pessoas do ArrayList de objetos do tipo Pessoa criado no item a;
* Exibir o nome e a idade da pessoa mais velha;
* Exibir o nome da  pessoa mais nova;
* Exibir a média de idade das pessoas;
* Numa classe que tem o método main, execute todos os métodos da classe OperaPessoa.
